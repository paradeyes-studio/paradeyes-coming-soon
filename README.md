# Paradeyes Coming Soon

Landing temporaire Paradeyes Agency en ligne jusqu'au lancement du site complet en juin 2026.

## Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Fonts: Instrument Serif + Inter (via `next/font/google`)

## Développement

```bash
npm install
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000).

## Build production

```bash
npm run build
npm run start
```

## Déploiement

Déployée sur Vercel, reliée au domaine `paradeyesagency.com` via DNS OVH.
